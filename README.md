## 一、项目风格 :watermelon:

手写机器学习的算法，以最简单为标准，此开源项目中的个个算法之间尽量保证独立性，帮助初学者快速根据原理实现 Machine Learning 中的算法。

个人在学习机器学习中的感受，老师讲课感觉还行，不是很难，等到自己实践时，只会敲一个Hello，Wolrd:cold_sweat:。
